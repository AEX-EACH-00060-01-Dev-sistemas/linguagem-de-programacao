# ESTRUTURAS DE CONDICIONAIS
Pense na situação: você está no seu carro e precisa escolher entre qual rua virar em um cruzamento. Você verifica seu GPS: Ele fala para virar para direita? Se sim, vire a direita; se não, vire a esquerda. Isso é base de uma estrutura condicional: um condição que desdobra em dois caminhos. No nosso português, seria mais ou menos assim:
```
SE (Condição) ENTÃO
  AÇÃO
SE NÃO
  OUTRA AÇÃO
```
Em Python essa estrutura é representada usando o `if`. Veja o exemplo abaixo:
```python
x = 10
if x < 10:
  print("x é menor que 10")
print("Aqui está fora do IF")
```

Diferente de outras linguagens que você verá por aí, em Python, a condição não é delimitado por parênteses e o restante do código não fica delimitado por chaves. Ao invés disso, só se coloca a expressão da condição depois do `if` e no final usa dois (:). O Python identifica o que está dentro da condição através da identação (aquele espaço que se tem antes do código, normalmente dado pelo `Tab` do teclado). Esse padrão se repetirá em laços de repetição e funções.
\
Mas caso queira que algo só execute quando a condição for falso, então usa-se o `else` logo depois do código do `if`. Análise o exemplo:
```python
x = 11

if x < 10:
  print("x é menor que 10")
else:
  print("x é maior ou igual a 10")
```

O `else` cobre aqueles casos não cumpre a condição incial. Mas caso queremos verificar mais condições, como podemos fazer? Então aí usados as **condições aninhadas**, ou seja, colocamos condições dentro de condições! Olhe o exemplo abaixo:
```python
x = 7

if x < 10:
  if x >= 5:
    print("x é maior ou igual a 5")
  else:
    print("x é menor que 5")
  print("x é menor que 10")
else:
  if x < 15:
    print("x é menor que 15")
  else:
    print("x é maior ou igual a 15")
  
  print("x é maior ou igual a 10")

print("Fim do programa!")
```

Apartir daí, é possível colocar inifitos `if` e `else` dentro do programa.

## ELIF
O `elif` é uma forma resumida de digitar uma condição aninhada. Para usá-la, segue-se o padrão:
```python
# Este código
if condição:
  Código 1
else:
  if condição 2:
    Código 2

# É IGUAL AO CÓDIGO ABAIXO
if condição:
  Código 1
elif condição 2:
  Código 2
```

## MATCH CASE
O Match case (o switch case, em outras linguagens - como C) é o um caso muito especial. Imagina que você, por ventura, precisa que, para cada mês do ano, apareça uma mensagem, usando o `if` fica mais ou menos assim:
```
SE janeiro ENTÃO
  mensagem 1
SENÃO SE fevereiro ENTÃO
  mensagem 2
SENÃO SE março ENTÃO
  mensagem 3
...
...
...
```

Isso ia ficar muito difícil de ler, até bastante trabalhoso! O match evita esses casos! Ele funciona basicamante assim:
```
PARA valor:
  CASO SEJA valor1 FAÇA:
    ação 1
  CASO SEJA valor2 FAÇA:
    ação 2
...
...
...
```

Isso pode parace muito trabalho, mas em diversas linguagens acaba sendo muito curto. Abaixo vejamos um exemplo em Python:
```python
x = "janeiro"

match x:
  case "janeiro":
    print("É Janeiro!")
  case "fevereiro":
    print("É Fevereiro!")
  case "março"
    print("É Março!")
```

Agora fica muito mais legivel de ler, em que o `match` vai pegar a variável x (ou uma qualquer) e verificar cada um dos caso. Mas e se quiser verificar apenas os três primeiros casos e o restante deixar uma resposta padrão? Bem, podemos fazer isso usando o *underline* (`_`):
```python
x = "janeiro"

match x:
  case "janeiro":
    print("É Janeiro!")
  case "fevereiro":
    print("É Fevereiro!")
  case "março"
    print("É Março!")
  case _:
    print("O restante dos meses")
```

Além disso, o Python tem um "macete" com `match`: na situação em que vários casos vão ter o mesmo código, é possível condensar em um caso só! Veja abaixo:
```python
mes = "janeiro"

match x:
  case "janeiro" | "fevereiro" | "março":
    print("Os três primeiros meses do ano!")
  case "outubro" | "novembro" | "dezembro":
    print("Os três últimos meses do ano!")
  case _:
    print("Meses restantes!")
```

AINDA HÁ MAIS! Junto do `case` é possível colocar um `if` em casos específicos:
```python
mes = 5
dia = 4
match dia:
  case 1 | 2 | 3 | 4 | 5 if mes == 4:
    print("Dia da semana em Abril")
  case 1 | 2 | 3 | 4 | 5 if mes == 5:
    print("Dia da semana em Maio")
  case _:
    print("No match")
```
